  • metacharacters, special sequences and sets

    metacharacters:

    • special characters in regex that have a special meaning
      • used to match one or more characters or character sets
    CharacterDescriptionExample
    .Matches any single character except newlinet.p matches tip, top, tap, etc.
    *Matches zero or more occurrences of the preceding characterca*t matches ct, cat, caat, caaat, etc.
    +Matches one or more occurrences of the preceding characterca+t matches cat, caat, caaat, etc.
    ?Matches zero or one occurrence of the preceding charactercolou?r matches color and colour
    ^Matches the start of the string^hello matches hello at the start of a string
    $Matches the end of the stringworld$ matches world at the end of a string
    ANCHORS: Do not match with specific characters, but specific positions within a string
    []Matches any character within the brackets[aeiou] matches any vowel
    [^]Matches any character not within the brackets[^aeiou] matches any consonant
    ()Groups patterns together(ca)t captures cat as a group
    |Matches either the pattern to the left or the pattern to the rightcat|dog matches cat or dog

    special sequences:

    • escape sequences used to match specific types of characters or character classes
    CharacterDescriptionExample
    \AMatches the start of the string, regardless of multiline mode"\Ahello" matches "hello" in "hello world", but not in "world hello"
    \bMatches the empty string at the beginning or end of a wordr“\bHa” matches the first “Ha” in “HaHa” but not the second “Ha”
    \BMatches the empty string not at the beginning or end of a wordr”\bHa” matches the second “Ha” in “HaHa” but not the first “Ha”
    \dMatches any decimal digit; equivalent to the character class [0-9]r"\d+" matches one or more decimal digits
    \DMatches any non-digit character; equivalent to the character class [^0-9]r"\D+" matches one or more non-digit characters
    \sMatches any whitespace character; equivalent to the character class [ \t\n\r\f\v]r"\s+" matches one or more whitespace characters
    \SMatches any non-whitespace character; equivalent to the character class [^ \t\n\r\f\v]r"\S+" matches one or more non-whitespace characters
    \wMatches any alphanumeric character; equivalent to the character class [a-zA-Z0-9_]r"\w+" matches one or more alphanumeric characters
    \WMatches any non-alphanumeric character; equivalent to the character class [^a-zA-Z0-9_]r"\W+" matches one or more non-alphanumeric characters

    • a set of characters inside a piar of square brackets with a special meaning
    SetDescriptionExample
    [arn]Returns a match where one of the specified characters (a, r, or n) is present"The cat sat on the mat." → Matches "a" in "cat"
    [a-n]Returns a match for any lower case character, alphabetically between a and n"The quick brown fox" → Matches "c" in "quick"
    [^arn]Returns a match for any character EXCEPT a, r, and n"The cat sat on the mat." → Matches "u" in "sat"

    • match() method
      • check whether a given pattern matches at the beginning of a string
    import re

sentence = 'Start a sentence and then bring it to an end'

pattern = re.compile(r'Start')

matches = pattern.match(sentence)

print(matches)

#OUTPUT:
<re.Match object; span=(0, 5), match='Start'>

    • search() function
      • searches the string from the beginning and stops as soon as it finds the first match for the pattern
      • does not check for any subsequent matches in the same string
    import re

sentence = 'Start a sentence and then bring it to an end'

pattern = re.compile(r'and')

matches = pattern.search(sentence)

print(matches)

#OUTPUT:
<re.Match object; span=(17, 20), match='and'>
  • flags
    • flags are optional parameters that modify how pattern matching is performed
    • can be passed as an argument to functions, or set within the pattern string itself

    • re.IGNORECASE / re.I
      • ignore case when matching
    import re

sentence = 'Start a sentence and then bring it to an end'

pattern = re.compile(r'start', re.I)

matches = pattern.search(sentence)

print(matches)

#OUTPUT:
<re.Match object; span=(0, 5), match='Start'>

    • re.MULTILINE / re.M
      • allows ^and $ to match the beginning and end of each line in a multiline string
    import re

text = '''Start a sentence and then bring it to an end
start a new line and start again
start the third line but not the last'''

pattern = re.compile(r'^start', re.MULTILINE)
matches = pattern.findall(text)

print(matches)

#OUTPUT:
['Start', 'start', 'start']

    • re.VERBOSE / re.X
      • allow writing RegEx in a more readable and organised way
      • whitespace characters inside pattern are ignored, except when they are escaped or within a character class
      • allows for use of comments #, ignored by the RegEx engine
    import re

pattern = re.compile(r'''
    ^              # match start of string
    \d{3}          # match 3 digits
    -              # match a hyphen
    \d{2}          # match 2 digits
    -              # match a hyphen
    \d{4}          # match 4 digits
    $              # match end of string
''', re.X)

    • re.ASCII
      • restricts character matching to ASCII characters only, even if the input string contains non-ASCII characters
    import re

sentence = 'Süßigkeiten und Leckereien'

pattern = re.compile(r'\w+', re.ASCII)

matches = pattern.findall(sentence)

print(matches)

#OUTPUT:
